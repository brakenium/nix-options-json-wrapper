# Kiji Proxy Home Assistant Add-on Repository

This repository contains a Home Assistant add-on for running Dataiku Kiji Privacy Proxy.

## Add-on path

- `kiji_proxy`

## Purpose

Run Kiji in Home Assistant and route LLM traffic through it so provider requests
(OpenAI, Anthropic, Gemini, Mistral, and compatible endpoints) are proxied through
Kiji for PII masking.
